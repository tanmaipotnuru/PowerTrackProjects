# LEXICAL ANALYZER PROJECT

**Name:** Potnuru Tanmai  
**Batch Number:** 25028

## Project Description
A C-based Lexical Analyzer that scans a C source file and identifies different lexical tokens such as keywords, identifiers, operators, special symbols, constants, and strings. The project performs tokenization and displays the token type for each lexeme.

## Features
- Identifies C keywords
- Detects identifiers
- Recognizes operators
- Identifies special symbols
- Detects numeric constants
- Detects string literals
- Reads input from a C source file
